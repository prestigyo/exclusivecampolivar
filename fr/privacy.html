2. Quelles données nous collectons

  • Données du formulaire de contact : le nom, l'adresse email, le numéro de téléphone et le message que vous soumettez via tout formulaire du Site.
  • Données analytiques : si — et seulement si — vous acceptez les cookies analytiques, Google Analytics collecte des données d'usage pseudonymisées (pages consultées, localisation approximative, appareil et navigateur). Voir notre Politique de Cookies.

Nous ne collectons pas sciemment de catégories particulières de données, et nous vous demandons de ne pas envoyer d'informations personnelles sensibles via les formulaires.

3. Pourquoi nous les utilisons, et notre base légale

  • Pour répondre à votre demande et fournir le service de publication de biens — base légale : votre consentement et/ou les démarches entreprises à votre demande avant un contrat (art. 6.1.a et 6.1.b du RGPD).
  • Pour comprendre et améliorer le Site via l'analytique — base légale : votre consentement (art. 6.1.a du RGPD), donné via le bandeau cookies.

4. Avec qui nous les partageons

Nous travaillons avec des prestataires de confiance (« sous-traitants ») qui agissent selon nos instructions :

  • Formspree (envoi de formulaires) — traite votre soumission pour nous la transmettre par email. Les données peuvent être traitées aux États-Unis dans le cadre de garanties appropriées.
  • Google Analytics (Google Ireland Ltd.) — uniquement si vous acceptez les cookies analytiques.
  • GitHub Pages (hébergement) — héberge les fichiers statiques du Site.

Nous ne vendons pas vos données personnelles. Certains prestataires peuvent transférer des données hors de l'EEE ; dans ce cas, le transfert repose sur les clauses contractuelles types de l'UE ou sur une décision d'adéquation.

5. Combien de temps nous les conservons

Nous conservons les données de votre demande uniquement le temps nécessaire pour la traiter et pour la relation qui en résulte, puis selon les obligations légales applicables. Les données analytiques sont conservées selon la période configurée dans Google Analytics.

6. Vos droits

Vous avez le droit d'accéder à vos données, de les rectifier, de les supprimer, d'en limiter ou de vous opposer au traitement, ainsi qu'à la portabilité des données. Pour exercer l'un de ces droits, écrivez à info@alcalahills.com. Vous pouvez retirer votre consentement à tout moment, sans affecter la licéité du traitement antérieur.

Si vous estimez que vos droits n'ont pas été respectés, vous pouvez déposer une réclamation auprès de l'Agence espagnole de protection des données (AEPD), www.aepd.es.

7. Sécurité

Le Site est servi en HTTPS et nous appliquons des mesures techniques et organisationnelles raisonnables pour protéger vos données. Aucune méthode de transmission sur Internet n'est cependant totalement sécurisée.
